# Chapitre 08 - BookShelf

## Description
Application Java de gestion d'une étagère à livres.
Elle permet d'ajouter des livres et de consulter la liste
de manière sécurisée grâce à une liste non modifiable.

## Fonctionnalités
- Ajouter un ou plusieurs livres
- Consulter la liste des livres
- Liste protégée contre les modifications externes
- Tests JUnit (tous passent)

## Classes
- **BookShelf** : classe principale qui gère l'étagère à livres
- **BookShelfSpec** : classe de tests JUnit avec 4 tests
